EduTest: система тестирования на Blazor + C# + MSSQL (вводная статья)

23 Nov 2025

EduTest — это пет-проект, цель которого — создать современную систему тестирования знаний школьников. Проект предназначен для учителей и учеников: учителя могут создавать тесты, управлять вопросами и отслеживать результаты, а ученики — проходить тесты, получать оценки и анализировать свои результаты.

Этот проект помогает мне отточить навыки работы со стеком:

  • Blazor Server — для фронтенда и интерактивного UI
  • C# — основной язык разработки
  • MSSQL + EF Core — база данных и ORM

Зачем этот проект

  • Углубиться в архитектуру приложений на C# и .NET
  • Научиться строить удобные админ- и пользовательские интерфейсы в Blazor
  • Попрактиковаться в проектировании БД и работе с Entity Framework Core
  • Зафиксировать весь процесс разработки в виде серии статей для блога
  • Создать проект, который можно расширять и улучшать со временем

Роли в системе

Для корректной работы и разграничения доступа в проекте предусмотрены три роли:

  1. Администратор
  • Управляет всеми пользователями (учителями и учениками)
  • Настраивает глобальные параметры системы
  • Контролирует корректность предметов, категорий и уровней сложности
  • Имеет полный доступ к данным и функционалу системы
  1. Учитель
  • Создаёт и редактирует тесты
  • Управляет банком вопросов
  • Просматривает результаты своих учеников
  • Настраивает учебные группы и тестовые параметры
  1. Ученик
  • Проходит тесты, доступные ему в системе
  • Получает результаты и статистику
  • Имеет доступ к истории своих попыток

Основные модули системы

Система EduTest будет состоять из следующих модулей:

  • Управление пользователями (администратор и роли)
  • Админ-панель (глобальное управление системой)
  • Панель учителя (создание тестов и вопросов)
  • Интерфейс ученика (прохождение тестов и просмотр результатов)
  • Аналитика и отчёты (статистика по результатам)
  • Безопасность и аутентификация (разграничение доступа)

Что будет в серии

Читатель узнает, как:

  • проектируются данные для системы тестирования
  • организовать архитектуру проекта для расширяемости
  • создать удобную админ-панель и панель учителя
  • реализовать интерфейс для учеников
  • хранить результаты и строить аналитику
  • роли пользователей предусмотрены и как работает авторизация
  • проект можно улучшать и развивать

Для кого эта серия

  • Начинающим и средним разработчикам C# и .NET
  • Тем, кто изучает Blazor и хочет делать интерактивные приложения
  • Людям, создающим свои пет-проекты
  • Тем, кто хочет увидеть процесс разработки от идеи до готового продукта
