Устанавливаем .NET Core в Ubuntu 22.04

03/04/2023

В этом уроке рассматривается процесс установки .NET Core на Ubuntu 22.04 LTS.

Предположим, что у читателя есть хотя бы базовые знания о Linux, и умеет он работать с командной строкой или в терминале, например Shell. Установка довольно проста и предполагается, что читатель работает под учетной записью root, в противном случае может потребоваться добавлять ‘sudo’ к командам, чтобы получить привилегии суперпользователя.

В этом уроке мы пройдем поэтапно весь процесс установки .NET Core на Ubuntu 22.04 (Jammy Jellyfish). Читатель может следовать тем же инструкциям для Ubuntu 20.04, 18.04, 16.04 и любого другого дистрибутива на базе Debian, такого как Linux Mint.

Step 1. Включаем Microsoft PPA

Первым делом включаем репозиторий Microsoft Apt в Ubuntu. Команда Microsoft предоставляет пакет Debian для настройки PPA в ОС Ubuntu. Запускаем терминал в Ubuntu и настроим Microsoft PPA, введя следующие команды:

wget https://packages.microsoft.com/config/ubuntu/22.04/packages-microsoft-prod.deb 
sudo dpkg -i packages-microsoft-prod.deb

Вышеприведенные команды создадут файл /etc/apt/sources.list.d/microsoft-prod.list с необходимой конфигурацией.

Теперь уже можно начать процесс установки .NET Core в Ubuntu.

Step 2. Устанавливаем .NET Core в Ubuntu

Первым делом мы должны обновить все пакеты системы, введя команды в терминале:

sudo apt update
sudo apt upgrade

И устанавливаем все необходимые компоненты .NET Core в Ubuntu, выполнив следующие команды:

  • Install the .NET Core SDK
sudo apt update
sudo apt install apt-transport-https
sudo apt update
sudo apt install dotnet-sdk-7.0
  • Install the ASP.NET Core runtime
sudo apt update
sudo apt install apt-transport-https
sudo apt update
sudo apt install aspnetcore-runtime-7.0
  • Install the .NET Core runtime
sudo apt update
sudo apt install apt-transport-https
sudo apt update
sudo apt install dotnet-runtime-7.0

Нажимаем клавишу “y” при каждом вводе, запрашиваемого программой установки.

Вот и все, мы успешно установили все необходимые компоненты .NET Core в систему Ubuntu.

Step 3. Проверяем версию .NET Core

После успешной установки мы можем использовать утилиту командной строки dotnet для проверки установленной версии .NET Core в нашей системе. Чтобы проверить версию .NET, введем следующую команду:

dotnet --version

Step 4. Создаем простое приложение .NET Core

Давайте создадим простое консольное приложение .NET Core в Ubuntu, написав следующую команду:

dotnet new console -o MyApp

Это создаст каталог с именем “MyApp”, содержащий консольное приложение .Net Core, в текущем каталоге. Можем перейти в этот каталог и начать работу со своим приложением.

cd  MyApp

Внеся свои изменения в код приложения, выполним нижеприведенную команду, чтобы запустить наше приложение.

dotnet run

Поздравляем! Мы успешно установили и протестировали .Net Core. Спасибо, что воспользовались этим уроком по установке Microsoft .Net Core в ОС Ubuntu 22.04 Jammy Jellyfish. Для получения дополнительной помощи или полезной информации рекомендуется посетить официальный сайт .Net Core.

Step 4. Удаляем или деинсталлируем .Net Core

В случае, если .Net Core больше нам не нужен, то можем его удалить его из системы с помощью следующих команд.

sudo apt remove --purge dotnet-sdk-7.0 dotnet-runtime-7.0

А также удаляем неиспользуемые пакеты, установленные в качестве зависимостей:

sudo apt auto-remove

Заключение

В этом уроке мы научились устанавливать .Net Core SDK и Runtime в Ubuntu 22.04 LTS (Jammy Jellyfish).
