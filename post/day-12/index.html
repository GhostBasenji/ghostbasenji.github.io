#12. Ввод и вывод в консольном приложении

26/11/2023

Иногда, например для научных расчетов, требуется организовать какой-нибудь самый простой ввод данных, выполнить, м.б., весьма сложную математическую обработку введенных данных и оперативно вывести на экран результат вычислений. Такая же ситуация возникает и тогда, когда большая программа отлаживается по частям. И для отладки вычислительной части совершенно не важен сервис при вводе данных.

Можно по-разному организовать такую программу, в т.ч. программируя консольное приложение. Давай напишем консольное приложение, которое приглашает пользователю ввести два числа, складывает их и выводит результат вычислений в консоль.

Для этого запускаем VS 2022, кликаем по Create a new project…, далее выбираем шаблон Console App, задаем имя решения - Sum. После того, как все сделали, попадаем сразу на вкладку программного кода.

Как видим, здесь VS 2022 оставила одну строку, потому что весь остальной код скрыт. Но это вполне работоспособная программа, если запустим нашу программу, то в консоли появится строка «Hello, World!».

Итак, напишем наш код:

// Программа организует ввод двух чисел, их сложение и вывод суммы на консоль
Console.Title = "Складываем два числа:";
Console.BackgroundColor = ConsoleColor.Cyan;  // цвет фона
Console.ForegroundColor = ConsoleColor.Black; // цвет текста
Console.Clear();

// Ввод первого слагаемого:
Console.WriteLine("Введите первое слагаемое:");
var Perem = Console.ReadLine();
// Преобразование строковой переменной в число:
var X = Single.Parse(Perem);

// Ввод второго слагаемого:
Console.WriteLine("Введите второе слагаемое:");
Perem = Console.ReadLine();
var Y = Single.Parse(Perem);

var Z = X + Y;

Console.WriteLine("Сумма = {0} + {1} = {2}", X, Y, Z);

// Звуковой сигнал частотой 1000 Гц и длительностью 0,5 секунд:
Console.Beep(1000, 500);

// Приостановить выполнение программы до нажатия какой-нибудь клавиши:
Console.ReadKey();

Обычно консольное приложение выполняется в окне на черном фоне. Чтобы как-то украсить традиционно черное окно консольного приложения, установим цвет фона окна BackgroundColor сине-зеленым (Cyan), а цвет символов, выводимых на консоль, черным (Black). Выводим строки в окно консоли методом WriteLine, а для считывания строки символов, вводимых пользователем, используем метод ReadLine. После ключевого слова var, названия переменной и оператора присваивания мы не указали тип переменной, поскольку компилятор сам назначит ей тип соответственно возвращаемому значению функции ReadLine(). Для преобразования строки символов, введеных пользователем, в числовое значение типа Single, используем метод Parse. Тип данных Single применяется тогда, когда число, записанное в переменную, может иметь целую и дробную части. Переменная типа Single занимает 4 байта.

После вычисления суммы необходимо вывести результат вычислений из оперативной памяти на экран. Для этого воспользуемся форматированным выводом в фигурных скобках метода WriteLine объекта Console:

Console.WriteLine("Сумма = {0} + {1} = {2}", X, Y, Z);

Затем выдаем звуковой сигнал Beep, сигнализирующий окончание процедуры и вывод на экран результатов вычислений. Последняя строка в программе Console.ReadKey() предназначается для приостановки выполнения программы до нажатия любой клавиши. Если не добавить эту строку, то окно с командной строкой появится и исчезнет мгновенно, и пользователь не сможет увидеть вывод результатов выполнения. Теперь посмотрим на результат работы консольного приложения, нажав клавишу F5.

При организации научных расчетов или в ситуации, когда необходимо отладить расчетную часть большой программы, когда сервис при вводе данных вообще не имеет значения, можно просто присваивать значения переменным при их объявлении. Технологичным является вариант, когда данные записываются в текстовый файл, а в программе предусмотрено чтение текстового файла в оперативную память.

Исходный код этого примера можно посмотреть, открыв решение Sum.sln в папке Sum.
